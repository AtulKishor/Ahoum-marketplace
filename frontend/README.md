# Full Stack Marketplace App

## Tech Stack
- Backend: Django, Django REST Framework, JWT
- Frontend: React (Vite)
- Auth: Google OAuth + JWT
- Containerization: Docker

## Features
- User & Creator roles
- JWT authentication
- Google OAuth login
- Session listing & booking
- Creator dashboard to manage sessions
- User dashboard to view bookings

## Run Locally (Docker)

```bash
docker-compose up --build
